
# Sociogram


## Built using

- ReactJS
- Redux Toolkit
- React Router
- Tailwind CSS


## Features

* User Feed
* Explore Feeds
* Latest and trending User Feed
* Follow/Unfollow
* Bookmark post
* Like/Unlike post
* Update user profile
* Authentication(SignUp , Login , Logout)
* Create/edit/delete post
* Create/edit/delete comment on post
* Responsive
